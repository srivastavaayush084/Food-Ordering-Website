# 🍔 Food Ordering Website

A modern and responsive **Food Ordering Website** built using **React.js** and **Vite**. The application provides an intuitive interface for users to browse food categories, explore menu items, manage their cart, and enjoy a seamless online food ordering experience.

## 🔗 Live Demo

* 🌐 **Website:**https://food-ordering-website-92em.vercel.app/
* 💻 **GitHub Repository:** https://github.com/srivastavaayush084/Food-Ordering-Website

## 🚀 Features

* 🍕 Browse food items by category
* 🔍 Search food items
* 🛒 Add and remove items from the shopping cart
* 💰 Dynamic cart total calculation
* 📱 Fully responsive design
* ⚡ Fast performance powered by Vite
* 🎨 Modern and clean UI

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3
* React Router DOM
